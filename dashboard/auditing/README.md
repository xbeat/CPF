# CPF Auditing Dashboard

## Overview

This is the **Auditing Dashboard** which combines the full CPF Auditing Dashboard with the **refactored Client**.

## What's Inside

This directory contains:

### Dashboard Components
- `index.html` - Main dashboard interface
- `dashboard.js` - Dashboard logic and functionality
- `styles.css` - Dashboard styles
- `category-descriptions.json` - Category metadata
- `reference_guide_*.json` - Reference guides (multilingual)
- `scripts/` - Additional dashboard scripts

### Client (Refactored)
- `client-integrated.js` - Refactored client
- `client-integrated.css` - Client styles
- `README-CLIENT.md` - Client documentation
- `README-CLIENT-PYTHON.md` - Python integration guide

### Testing Tools
- `index-client-test.html` - Client test page
- `test-runner.html` - Automated test runner
- `test-suite.js` - Client test suite
- `run-tests.js` - Test execution scripts
- `run-tests-simple.js` - Simplified test runner

## Purpose

This dashboard allows you to:
- **Test the refactored client** in a full dashboard environment
- **Run tests** while using the dashboard
- **Validate** all client functionality in a production-like setting

## Usage

1. Open `index.html` in a browser to use the full dashboard
2. Open `index-client-test.html` for quick client testing
3. Open `test-runner.html` to run the automated test suite

## Features

- Uses the **refactored client** with modern ES6+ architecture
- Improved code organization and maintainability
- Enhanced testing capabilities
- Complete dashboard functionality
