# Emerging Threats Analysis (CPF Perspective)

This directory contains analysis of emerging cybersecurity threats through the lens of the Cybersecurity Psychology Framework.

## Purpose

To analyze new and evolving cyber threats by examining the psychological vulnerabilities they exploit, using the CPF taxonomy.

## Structure

Each threat is analyzed in its own subdirectory with:
- Threat description and technical analysis
- CPF vulnerability mapping
- Psychological exploitation mechanisms
- Recommended mitigations

## Current Analyses

- `2025-agentic-ai-systems/` - Agentic AI systems and psychological vulnerabilities
