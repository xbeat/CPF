# Articoli di Ricerca CPF

Questa directory contiene articoli di ricerca relativi al Cybersecurity Psychology Framework.

## Contenuti

Articoli accademici e tecnici che esplorano vari aspetti delle vulnerabilità psicologiche nei contesti di cybersecurity.

## Argomenti

- Contributi psicoanalitici alla cybersecurity
- Psicologia cognitiva nei contesti di sicurezza
- Vulnerabilità psicologiche specifiche dell'AI
- Dinamiche di gruppo nelle operazioni di sicurezza
- Case study e validazione empirica

## Formato

Gli articoli sono tipicamente in formato PDF o LaTeX per pubblicazione accademica.
