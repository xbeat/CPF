# CPF Documentation

This directory contains comprehensive documentation for the **Cybersecurity Psychology Framework (CPF)** - the first psychoanalytic-cognitive model for pre-cognitive security vulnerabilities.

## The 10 Vulnerability Categories

The CPF Framework identifies 100 indicators across these 10 psychological domains:

| Category | Key Concept | Example Vulnerability |
|----------|-------------|----------------------|
| **1. Authority-Based** | Milgram obedience | CEO fraud susceptibility |
| **2. Temporal** | System 1 heuristics | Urgency-induced security bypass |
| **3. Social Influence** | Cialdini's principles | Reciprocity exploitation |
| **4. Affective** | Klein-Bowlby affect theory | Fear-based decision paralysis |
| **5. Cognitive Overload** | Miller's cognitive limits | Alert fatigue desensitization |
| **6. Group Dynamics** | Bion's basic assumptions | Groupthink security blind spots |
| **7. Stress Response** | Selye's stress theory | Acute stress impairment |
| **8. Unconscious Processes** | Jungian shadow | Shadow projection onto attackers |
| **9. AI-Specific** | Anthropomorphism | Over-trust in AI systems |
| **10. Critical Convergent States** | System theory | Perfect storm conditions |

## Key Framework Features

- **100 Indicators** across 10 vulnerability categories
- **Ternary Scoring**: Green/Yellow/Red risk assessment
- **Attack Vector Mapping** to known exploit patterns
- **Privacy-Preserving**: Aggregate analysis, no individual profiling
- **Evidence-Based**: Built on 50+ years of psychological research
- **Interactive Tools**: Multi-language support (English & Italian)

## Scientific Foundation

The CPF Framework integrates:

- **Psychoanalytic Theory**: Bion's Basic Assumptions, Kleinian Splitting, Jungian Shadow
- **Cognitive Psychology**: Kahneman's System 1/System 2, Cialdini's Influence Principles
- **Neuroscience**: Libet's pre-cognitive decision research (300-500ms latency)
- **Social Psychology**: Milgram's obedience studies, groupthink dynamics
- **Stress Research**: Selye's stress theory, acute/chronic stress responses
