# CPF Translation Resources

This directory contains resources and workflows for translating CPF documentation.

## Purpose

Centralized location for translation guidelines, terminology databases, and translation workflows.

## Languages

Current translation efforts:
- English (en-US) - Original
- Italian (it-IT) - In progress

## Guidelines

When translating CPF materials:
- Maintain literal translation without structural modifications
- Preserve all LaTeX commands and mathematical formulas
- Do not translate common English technical terms (framework, software, security, cloud, etc.)
- Maintain exact text length where possible
- Keep all bibliographic references in original language
