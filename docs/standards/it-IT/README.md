# Documentazione Standard CPF - Italiano (it-IT)

Questa directory contiene la documentazione ufficiale in italiano del Cybersecurity Psychology Framework (CPF).

## Contenuti

### Standard Principali
- **CPF 27001 Requisiti.tex** - Requisiti per il Sistema di Gestione delle Vulnerabilità Psicologiche (PVMS)
- **CPF Annex A - Guida alla Mappatura dei Controlli.tex** - Integrazione con ISO 27002, NIST CSF, CIS Controls
- **CPF Linee Guida di Audit v1.0.tex** - Linee guida per condurre audit CPF-27001
- **CPF Guida di Avvio Rapido - Inizia in 90 Giorni.tex** - Guida all'implementazione in 90 giorni
- **CPF Modello di Score e Maturità v1.0.tex** - Metodologia di assessment quantitativo
- **CPF-Schema-di-Certificazione.tex** - Processo e requisiti di certificazione

### Materiali di Formazione
Nella sottocartella `Training/`:
- CPF3 101 Progetto di Formazione
- CPF3 201 Progetto di Formazione
- CPF3 301 Progetto di Formazione
- CPF-401-Progetto-di-Formazione

### Accordi di Licenza
Nella sottocartella `Licensing Agreement/`:
- CPF3 Accordo di Certificazione Professionale Individuale
- CPF3 Accordo di Certificazione Organizzativa
- CPF3 Provider di Servizi Autorizzato
- CPF3 Accordo di Servizi Professionali
- CPF3 Accordo di Licenza di Schema
