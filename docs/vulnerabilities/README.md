# CPF Vulnerabilities Documentation

This directory contains detailed documentation of the CPF vulnerability categories.

## Structure

- `en-US/` - English documentation of CPF vulnerabilities
- `it-IT/` - Italian translations of CPF vulnerabilities

## Contents

Documentation for all 10 CPF vulnerability categories:

- [1.x] Authority-Based Vulnerabilities
- [2.x] Temporal Vulnerabilities
- [3.x] Social Influence Vulnerabilities
- [4.x] Affective Vulnerabilities
- [5.x] Cognitive Overload Vulnerabilities
- [6.x] Group Dynamic Vulnerabilities
- [7.x] Stress Response Vulnerabilities
- [8.x] Unconscious Process Vulnerabilities
- [9.x] AI-Specific Bias Vulnerabilities
- [10.x] Critical Convergent States

## Language Support

All documents are available in both English (en-US) and Italian (it-IT).
