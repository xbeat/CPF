# CPF Presentations

This directory contains presentation materials for the Cybersecurity Psychology Framework.

## Structure

Presentations are organized by CPF vulnerability category:

- `1.x-authority/` - Authority-Based Vulnerabilities presentations
- `2.x-temporal/` - Temporal Vulnerabilities presentations
- `3.x-social/` - Social Influence Vulnerabilities presentations

## Format

Presentations are typically available in PDF and source formats for easy customization and distribution.

## Usage

These materials are designed for training, conferences, and educational purposes related to the CPF framework.
