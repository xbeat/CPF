# Temporal Vulnerabilities Presentations

Presentation materials for CPF Category [2.x] - Temporal Vulnerabilities.

## Category Overview

Temporal vulnerabilities arise from how time pressure, deadlines, and temporal patterns affect security-relevant decisions.

## Topics Covered

- Urgency-induced security bypass
- Time pressure cognitive degradation
- Deadline-driven risk acceptance
- Present bias in security investments
- Temporal exhaustion patterns

## Presentation Files

This directory contains slides and materials for presenting temporal vulnerabilities in various contexts.
