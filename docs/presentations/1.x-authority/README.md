# Authority-Based Vulnerabilities Presentations

Presentation materials for CPF Category [1.x] - Authority-Based Vulnerabilities.

## Category Overview

Authority-based vulnerabilities emerge from human tendencies to obey perceived authority figures, as demonstrated in Milgram's experiments (1974).

## Topics Covered

- Unquestioning compliance with authority
- Diffusion of responsibility in hierarchical structures
- Authority figure impersonation susceptibility
- Executive exception normalization
- Crisis authority escalation

## Presentation Files

This directory contains slides and materials for presenting authority-based vulnerabilities in various contexts.
