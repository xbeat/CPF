# Social Influence Vulnerabilities Presentations

Presentation materials for CPF Category [3.x] - Social Influence Vulnerabilities.

## Category Overview

Social influence vulnerabilities leverage fundamental human needs for social connection, consistency, and belonging.

## Topics Covered

- Reciprocity exploitation
- Commitment escalation traps
- Social proof manipulation
- Liking-based trust override
- Scarcity-driven decisions

## Presentation Files

This directory contains slides and materials for presenting social influence vulnerabilities in various contexts.
