# CPF Core Foundation Documentation

This directory contains the core foundation documentation for the **Cybersecurity Psychology Framework (CPF)**, organized by language.

## Overview

The core documentation provides comprehensive academic and technical materials that establish the theoretical and empirical foundations of CPF. These documents represent the primary reference materials for understanding, implementing, and validating the framework.

## Key Features

### The 10 Domains of Psychological Vulnerability

CPF identifies 10 core domains with 100 specific behavioral indicators:

1. **Authority & Obedience** - Unconscious compliance patterns
2. **Temporal Factors** - Time-pressure and urgency exploitation
3. **Social Influence** - Peer pressure, reciprocity, social proof
4. **Affect & Emotion** - Emotional states affecting decisions
5. **Cognitive Load** - Information overload and attention management
6. **Group Dynamics** - Collective behavior and organizational psychology
7. **Stress Response** - Fight-flight-freeze-fawn responses
8. **Unconscious Process** - Defense mechanisms and unconscious patterns
9. **AI Interaction** - Human-AI psychological dynamics
10. **Systemic Complexity** - Emergence and cascade effects
