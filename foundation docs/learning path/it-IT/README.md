# CPF Educational Framework - Italiano

Benvenuti nella documentazione educativa per il **Cybersecurity Psychology Framework (CPF)**.

## Informazioni su Questa Raccolta

Questa directory contiene il CPF Educational Framework, un curriculum strutturato progettato per introdurre, sviluppare e specializzare i discenti attraverso l'intero spettro della literacy in cybersecurity psicologica. A differenza dei tradizionali programmi di security awareness che assumono attori razionali modificabili attraverso il trasferimento di informazioni, questo approccio educativo riconosce che le decisioni di security avvengono sostanzialmente al di sotto della consapevolezza cosciente e che un'educazione efficace deve coinvolgere i processi pre-cognitivi, le dinamiche di gruppo e la complessa interazione tra intelligenza umana e artificiale.

## Il CPF Educational Framework

Il framework educativo comprende quattro moduli universali che formano uno scheletro concettuale invariante, modulato attraverso quattro livelli di sviluppo:

### Quattro Moduli Universali

1. **"Non Decidi Tu"** - Processi pre-cognitivi e decision-making inconscio
2. **"Come Ti Fregano"** - Tecniche di sfruttamento psicologico e vulnerabilità
3. **"Il Gruppo Pensa Per Te"** - Dinamiche di gruppo e psicologia organizzativa
4. **"Tu e le Macchine"** - Interazione umano-IA e dinamiche psicologiche

### Quattro Livelli di Sviluppo

- **Livello Base** - Literacy psicologica di base per il pubblico generale
- **Livello Intermedio** - Comprensione più profonda per i practitioner di security
- **Livello Avanzato** - Conoscenza specializzata per i professionisti della security
- **Livello Specialistico** - Padronanza a livello esperto per ricercatori e architetti

## Filosofia Educativa

Questo curriculum rifiuta le assunzioni fondamentali della security awareness tradizionale:

- **Oltre i Modelli di Attore Razionale**: Le decisioni di security avvengono 300-500ms prima della consapevolezza cosciente
- **Focus Pre-Cognitivo**: Targeting dei processi automatici che dominano quelli deliberati
- **Integrazione delle Dinamiche di Gruppo**: Il comportamento collettivo emerge da basic assumption inconsce
- **Fondazione Empirica**: Basato su neuroscienze, economia comportamentale e teoria psicoanalitica

## Percorso di Apprendimento Progressivo

Il curriculum posiziona i paper fondamentali del CPF come waypoint progressivi:

1. **CPF Taxonomy** - Mappa di riferimento completa delle vulnerabilità psicologiche
2. **Dense Implementation Companion** - Specifica operativa e metodologia di assessment
3. **Intervention Framework** - Metodologia di remediation e gestione delle crisi
4. **Depth Paper** - Mentore teorico che accompagna i discenti durante tutto il loro viaggio

## Pubblici di Destinazione

### Pubblico Generale (Livello Base)
- Security awareness oltre le checkbox di compliance
- Comprensione delle tecniche di manipolazione psicologica
- Literacy pre-cognitiva di base

### Practitioner di Security (Livello Intermedio)
- Comprensione operativa dei fattori umani
- Implementazione della metodologia di assessment
- Integrazione con i programmi di security esistenti

### Professionisti della Security (Livello Avanzato)
- Assessment approfondito delle vulnerabilità psicologiche
- Gestione delle crisi e intervento
- Analisi delle dinamiche organizzative

### Ricercatori e Architetti (Livello Specialistico)
- Estensione e specializzazione del framework
- Metodologia di validazione della ricerca
- Profondità teorica e integrazione

## Obiettivi di Apprendimento Chiave

Progredendo attraverso questo curriculum, i discenti:

1. Comprenderanno che il decision-making cosciente è largamente illusorio nei contesti di security
2. Riconosceranno i 10 domini e i 100 indicatori di vulnerabilità psicologica
3. Valuteranno i processi pre-cognitivi in se stessi e nelle loro organizzazioni
4. Implementeranno interventi evidence-based per le vulnerabilità psicologiche
5. Navigheranno le dinamiche di interazione umano-IA con consapevolezza psicologica
6. Gestiranno eventi di crisi attraverso framework psicologici
