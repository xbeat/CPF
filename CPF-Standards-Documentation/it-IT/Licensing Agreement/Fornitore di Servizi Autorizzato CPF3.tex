\section*{PREMESSE}

CONSIDERATO CHE, l'OdC è autorizzato da CPF3 a certificare organizzazioni come Fornitori di Servizi Autorizzati;

CONSIDERATO CHE, l'Azienda fornisce servizi relativi a CPF e desidera la certificazione "Fornitore di Servizi Autorizzato CPF";

CONSIDERATO CHE, l'Azienda impiega professionisti CPF certificati e mantiene sistemi di gestione della qualità;

PERTANTO, le Parti convengono quanto segue:

\section{DEFINIZIONI}

\textbf{1.1 "Fornitore di Servizi Autorizzato"} indica un'azienda certificata per fornire servizi CPF e utilizzare la designazione "Fornitore di Servizi Autorizzato CPF".

\textbf{1.2 "Servizi CPF"} indica servizi professionali inclusi assessment, implementazione, formazione, monitoraggio e integrazione.

\textbf{1.3 "Personale Certificato"} indica individui con certificazioni CPF valide impiegati dall'Azienda.

\textbf{1.4 "Marchio ASP"} indica il marchio e logo "Fornitore di Servizi Autorizzato CPF".

\section{PROCESSO DI CERTIFICAZIONE}

\textbf{4.1 Domanda (Settimane 1-4):}

Presentare:
\begin{itemize}
\item Modulo di domanda
\item Lista personale certificato
\item Documentazione QMS
\item Framework privacy e audit
\item Policy etiche e registrazioni
\item Certificati assicurativi
\item Deliverable di esempio (3 oscurati)
\item Referenze clienti (5)
\item Organigramma
\item Tariffa di domanda
\end{itemize}

L'OdC conduce:
\begin{itemize}
\item Revisione documentale
\item Verifica del personale
\item Controllo referenze (minimo 3)
\item Valutazione QMS
\end{itemize}

\textbf{4.2 Audit (Settimane 6-8):}

Durata: 2-4 giorni

Attività:
\begin{itemize}
\item Riunione di apertura
\item Revisione QMS
\item Interviste al personale
\item Revisione file di progetto (3-5)
\item Valutazione infrastruttura
\item Valutazione privacy
\item Valutazione etica
\item Verifica assicurativa
\item Riunione di chiusura
\end{itemize}

Risultati: Conformità, NC Minore, NC Maggiore, Osservazione

Report: 15 giorni lavorativi

\textbf{4.3 Azioni Correttive:}

Se NC:
\begin{itemize}
\item Piano entro 30 giorni
\item Maggiori: Prima della certificazione
\item Minori: Entro 90 giorni dopo
\item Verifica
\end{itemize}

\textbf{4.4 Decisione (15 giorni):}

\textit{Concessa:}
\begin{itemize}
\item Certificato (elettronico 3 giorni, fisico 10)
\item Badge digitale
\item Inserimento prioritario in directory
\item Kit marketing
\item Accesso alle risorse
\item Programma di sorveglianza
\end{itemize}

\textit{Negata:}
\begin{itemize}
\item Spiegazione scritta
\item Guida alla risoluzione
\item Diritti di appello (30 giorni)
\item Opzione di riapplicazione
\end{itemize}

\section{SOSPENSIONE E REVOCA}

\textbf{7.1 Sospensione (max 180 giorni):}

Motivi:
\begin{itemize}
\item Sotto personale minimo
\item Decadenza assicurativa
\item Fallimento sorveglianza
\item Mancato pagamento tariffe
\item Reclami sotto investigazione
\item Violazione privacy
\item NC maggiore non corretta (90 giorni)
\item Soddisfazione cliente <3.5 (2 trimestri)
\end{itemize}

Effetto:
\begin{itemize}
\item Restrizione su nuovo uso del Marchio
\item Directory: "Sospeso"
\item Aggiungere "Status in Revisione"
\item Referral sospesi
\item 180 giorni per rimediare o revoca
\end{itemize}

\textbf{7.2 Revoca:}

Motivi:
\begin{itemize}
\item Mancata risoluzione
\item Violazioni privacy gravi
\item Frode/dichiarazioni false
\item Fallimenti sistematici di qualità
\item Violazioni etiche
\item Uso improprio persistente del Marchio
\item Perdita assicurazione >60 giorni
\item Dissoluzione/fallimento
\end{itemize}

Processo:
\begin{itemize}
\item Notifica scritta
\item 30 giorni per rispondere
\item Revisione indipendente
\item Decisione entro 45 giorni
\item Diritti di appello
\end{itemize}

Effetto:
\begin{itemize}
\item Cessazione immediata di TUTTO l'uso del Marchio
\item Rimozione dalla directory
\item Avviso pubblico (12 mesi)
\item Nessun rimborso
\item Riapplicazione: 2-5 anni o permanente
\item Deve notificare i clienti
\end{itemize}

\textbf{7.3 Recesso Volontario:}
\begin{itemize}
\item Preavviso 60 giorni
\item Nessun record negativo
\item Riapplicazione dopo 12 mesi
\item Semplificato se entro 24 mesi
\end{itemize}
