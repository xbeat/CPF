# Accordi di Licenza CPF

Questa directory contiene i template ufficiali di licenza e accordi CPF.

## Accordi Disponibili

### CPF3 Accordo di Certificazione Professionale Individuale
Per professionisti individuali che cercano la certificazione CPF.

### CPF3 Accordo di Certificazione Organizzativa
Per organizzazioni che perseguono la certificazione CPF-27001.

### CPF3 Provider di Servizi Autorizzato
Accordo per diventare un provider di servizi CPF autorizzato.

### CPF3 Accordo di Servizi Professionali
Template per incarichi di servizi professionali.

### CPF3 Accordo di Licenza di Schema
Accordo per la licenza dello schema di certificazione CPF.

## Utilizzo

Questi sono accordi template che dovrebbero essere rivisti da un consulente legale prima dell'uso.

## Generazione PDF

Per compilare gli accordi in PDF:
```bash
pdflatex "CPF3 Accordo di Certificazione Professionale Individuale.tex"
```
