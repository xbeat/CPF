# Materiali di Formazione CPF

Questa directory contiene i progetti ufficiali di formazione CPF.

## Corsi di Formazione Disponibili

### CPF3 101 Progetto di Formazione
**Livello:** Foundation
**Durata:** 2 giorni
**Pubblico di destinazione:** Professionisti della sicurezza, HR, chiunque sia nuovo a CPF
**Prerequisiti:** Nessuno

### CPF3 201 Progetto di Formazione
**Livello:** Intermedio
**Durata:** 3 giorni
**Pubblico di destinazione:** Analisti di sicurezza, consulenti
**Prerequisiti:** CPF3 101

### CPF3 301 Progetto di Formazione
**Livello:** Avanzato
**Durata:** 5 giorni
**Pubblico di destinazione:** Consulenti senior, ricercatori
**Prerequisiti:** CPF3 201

### CPF-401 Progetto di Formazione
**Livello:** Esperto
**Durata:** 10 giorni
**Pubblico di destinazione:** Sviluppatori di framework, professionisti avanzati
**Prerequisiti:** CPF3 301

## Generazione PDF

Per compilare i materiali di formazione in PDF:
```bash
pdflatex "CPF3 101 Progetto di Formazione.tex"
```
