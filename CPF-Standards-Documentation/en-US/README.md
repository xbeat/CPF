# CPF Standards Documentation - English (en-US)

This directory contains the official English documentation for the Cybersecurity Psychology Framework (CPF).

## Contents

### Core Standards
- **CPF 27001 Requirements.tex** - Requirements for Psychological Vulnerability Management System (PVMS)
- **CPF Annex A - Control Mapping Guide.tex** - Integration with ISO 27002, NIST CSF, CIS Controls
- **CPF Audit Guidelines v1.0.tex** - Guidelines for conducting CPF-27001 audits
- **CPF Quick Start Guide - Get Started in 90 Days.tex** - 90-day implementation guide
- **CPF Scoring and Maturity Model v1.0.tex** - Quantitative assessment methodology
- **CPF-Certification-Scheme.tex** - Certification process and requirements

### Training Materials
Located in `Training/` subdirectory:
- CPF3 101 Training Blueprint
- CPF3 201 Training Blueprint
- CPF3 301 Training Blueprint
- CPF-401-Training-Blueprint

### Licensing Agreements
Located in `Licensing Agreement/` subdirectory:
- CPF3 Individual Professional Certification Agreement
- CPF3 Organizational Certification Agreement
- CPF3 Authorized Service Provider
- CPF3 Professional Services Agreement
- CPF3 Scheme Licensing Agreement

## Available Languages

- **en-US** (English) - This directory
- **it-IT** (Italian) - Translated version available in ../it-IT/

## Generating PDFs

To compile LaTeX documents to PDF:
```bash
pdflatex "CPF 27001 Requirements.tex"
```
