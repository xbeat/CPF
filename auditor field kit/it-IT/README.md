# CPF Auditor Field Kit - Italiano

Strumenti di valutazione operativi per le vulnerabilità psicologiche in cybersecurity.

Questa cartella contiene i documenti di valutazione in italiano organizzati per dominio (1.x fino a 10.x). Ogni dominio include:
- **Foundation** (`*_foundation.md`): Analisi teorica approfondita con evidenze scientifiche
- **Operational** (`*_operational.md`): Protocolli di valutazione con metodologie di scoring
- **Field Kit** (`*_field_kit.md`): Assessment rapido (30 minuti) per identificazione veloce vulnerabilità

## Come usare

1. Inizia con il **Field Kit** per una valutazione rapida
2. Approfondisci con l'**Operational** per analisi composta
3. Consulta il **Foundation** per giustificazione scientifica e training

Per maggiori dettagli sulla struttura complessiva, vedi il [README principale](../README.md).
