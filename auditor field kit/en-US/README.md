# CPF Auditor Field Kit - English

Operational assessment tools for cybersecurity psychology vulnerabilities.

This folder contains assessment documents in English organized by domain (1.x through 10.x). Each domain includes:
- **Foundation** (`*_foundation.md`): In-depth theoretical analysis with scientific evidence
- **Operational** (`*_operational.md`): Assessment protocols with detailed scoring methodology
- **Field Kit** (`*_field_kit.md`): Rapid 30-minute assessment tools for quick vulnerability identification

## How to Use

1. Start with the **Field Kit** for rapid vulnerability identification
2. Deep-dive with **Operational** for comprehensive analysis
3. Reference **Foundation** for scientific justification and training

For complete framework details, see the [main README](../README.md).
